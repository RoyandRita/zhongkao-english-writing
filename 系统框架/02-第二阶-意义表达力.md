# 第二阶：意义表达力

## ——14种功能表达，让每句话都有"战术任务"

---

## 什么是"意义表达"？

中考作文的本质，不是"写一篇文章"，而是"完成一系列功能任务"。

题目说"请给出建议"→ 你需要一个**建议功能句**。
题目说"描述你的感受"→ 你需要一个**感受功能句**。
题目说"说明原因"→ 你需要一个**原因功能句**。

```
普通学生的写作方式：
"我想说什么就写什么"→ 句子之间没有功能意识 → 要点遗漏或重复

训练后的写作方式：
"这个要点需要什么功能？"→ 调用该功能的句型模板 → 精准完成
```

14种功能表达，覆盖了武汉中考书面表达中几乎所有要点要求的功能类型。每一种功能都有一张**微型训练卡**，结构完全相同，练完一种就掌握一种。

## 每张微型训练卡的统一结构

14张卡采用完全相同的结构设计，确保学生练完第一张卡后，拿到任何一张新卡都能独立使用：

```
┌─ 卡片X：[功能名称] ──────────────────────────┐
│                                               │
│  ① 功能定位                                  │
│     这个功能在文章什么位置用？解决什么要点？    │
│     → 30秒读懂                                │
│                                               │
│  ② 基础句型库（3个模板）                      │
│     中英对照，可直接填入任何话题使用           │
│     → 2分钟背熟，遮住英文看中文能说出来        │
│                                               │
│  ③ 升级句型库（2个提分模板）                  │
│     比基础版好在哪？什么条件下使用？           │
│     → 2分钟理解"好在哪里"                     │
│                                               │
│  ④ 常见错误警示（2个最高频错误）              │
│     错误版 vs 正确版对比 + 为什么错           │
│     → 1分钟提前避开                           │
│                                               │
│  ⑤ 分级练习                                  │
│     Lv1 单句翻译（3题）→ "我能写出来"         │
│     Lv2 情境回应（2题）→ "我能用对地方"       │
│     Lv3 段落嵌入（1题）→ "我能融入全文"       │
│     → 10-15分钟独立完成                       │
│                                               │
│  ⑥ 中考真题链接                              │
│     1道真实中考题的要點拆解，验证"这真的会考"  │
│     → 2分钟建立信心                           │
│                                               │
└───────────────────────────────────────────────┘
```

## 14种功能表达的"战术位置"速查

写作文时，学生可以根据"我现在在文章的哪个位置"，快速判断需要调用哪种功能：

```
文章位置              需要的主要功能            辅助功能
─────────────────────────────────────────────────────
P1 开头               引入主题                 F10 表达观点
                      建立场景                 F3 邀请（书信）
                                              F4 道歉（书信）
                                              F2 感谢（书信）

P2 中段（前半）       展开第一个核心内容        F6 描述事件
                                              F9 说明原因
                                              F10 表达观点

P2 中段（后半）       展开第二个核心内容        F6 描述事件
                      或深化论证               F8 举例支撑
                                              F11 对比转折
                                              F14 表达目的

P2→P3 过渡           完成论证弧度              F1 建议
                                              F5 劝说

P3 结尾               收束全文                 F12 总结升华
                                              F13 展望未来
                                              F7 描述感受
                                              F2 感谢（回扣）
```

## 三种功能组合模式（最高频）

大多数中考作文都可以归结为以下三种功能组合：

### 模式一：叙事模式（活动类/经历类）

```
F6 描述事件 → F6 描述事件 → F7 描述感受 → F12 总结升华
 （动作1）      （动作2）     （感受前有画面）  （从事件中提炼）
```

### 模式二：论证模式（观点类/建议类）

```
F10 表达观点 → F9 说明原因 → F8 举例支撑 → F1 建议 → F12 总结升华
 （立场）       （理由1）     （具体例子）  （建议）  （升华）
```

### 模式三：人际模式（书信类/发言稿类）

```
F3/F4 开场功能 → F6 描述事件 → F7 描述感受 → F1 建议/F5 劝说 → F13 展望未来
 （邀请/道歉）    （发生了什么） （我感受如何）  （给对方建议）   （美好画面）
```

## 训练的进阶标准

学生完成每张卡的训练后，对照以下标准自评：

```
★★★ 达标（可以前进）：
  □ Lv1的3题全部正确，无语法错误
  □ Lv2的2题功能使用正确（句型选对了）
  □ Lv3的1题能自然嵌入段落中

★★★★ 良好：
  □ 以上全部 +
  □ Lv2中使用了升级句型（而非只用基础句型）
  □ Lv3中段落逻辑通顺，无孤岛句

★★★★★ 优秀：
  □ 以上全部 +
  □ Lv3中使用了不止一种配套功能（如建议+原因同时出现）
  □ 能在不同话题间灵活迁移同一个句型模板
```

**重做规则**：低于★★★的卡片，隔天全部重做，直到达标。

---

## 14张卡片的推荐练习顺序

按难度和实用频率排列：

```
第一周（人际功能，最常用，上手快）：
  F1 建议 → F2 感谢 → F3 邀请 → F4 道歉 → F5 劝说

第二周（叙述功能，需要配合画面训练）：
  F6 描述事件 → F7 描述感受 → F8 举例支撑 → F9 说明原因

第三周（议论功能，逻辑要求最高）：
  F10 表达观点 → F11 对比转折 → F12 总结升华 → F13 展望未来 → F14 表达目的

每张卡约需25-30分钟（含Lv1-Lv3全部练习）。
```

---

> **本阶核心理念**：
>
> 学生不是"不会写英语"，而是"不知道该写什么功能的句子"。
>
> 14张卡训练完，学生拿到任何题目，看到任何要点，
> 大脑里会立刻响起："这个要点是'建议'功能，调用F1模板。"
>
> 这就是从"凭感觉写"到"有策略地写"的跨越。

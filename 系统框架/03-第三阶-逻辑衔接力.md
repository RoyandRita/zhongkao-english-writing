# 第三阶：逻辑衔接力

## ——让每句话"召唤"下一句，消灭孤岛句

---

## 核心问题

基础好的学生最容易出现的隐藏失分点：

```
表面上看：语法正确，词汇不错，要点齐全
实际上读：句子之间是"拼"在一起，不是"长"在一起

这就是"段落孤岛症"——
每句话单独拿出来都对，
但连在一起没有逻辑驱动力，
阅卷老师读起来"跳"。
```

---

## 核心工具：衔接三问

写完每一句话后，快速运行这三个问题：

```
═══════════════════════════════════════════

问①：上句说了什么？
     → 用一句话概括上句的核心信息。

问②：下句应该回应什么？
     → 读者读完上句后，脑子里自然产生什么疑问/期待？
     → 这个疑问/期待 = 下句的方向。

问③：用什么连接词？
     → 两句之间是什么逻辑关系？
     → 选出最精准的连接词（不只用First/Second/Third）。

═══════════════════════════════════════════
```

### 衔接三问实例

**上句**：Social media allows teenagers to connect with friends easily.

```
问①：上句说了什么？
     → 社交媒体让青少年容易和朋友联系。

问②：读者读完后自然想问什么？
     → 怎么联系？（举例）
     → 这样有什么好处？（深入说明）
     → 但是有什么问题？（转折）
     → 所以呢？（结果）
     
     选一个方向 → 我选"但是有什么问题"。

问③：两句之间是什么关系？用什么连接词？
     → 正面 → 负面 = 转折
     → 连接词：However

下句：However, spending too much time on it can leave them 
      feeling more isolated than connected.
```

## 四种逻辑驱动力

有了"衔接三问"的意识后，用四种具体驱动力模型来精准控制下句方向：

### 驱动力类型①：问题 → 答案

```
触发句（Statement）给出一个现象或结论
         ↓
驱动句（Question）将其转化为一个"需要回答的问题"
         ↓
答案句（Answer）给出解答
```

**示例**：

```
触发句：Reading every day is one of the best habits a student can develop.
         ↓ "但为什么很多学生做不到？"
驱动句：Yet for many students, finding the time to open a book 
         every single day is far harder than it sounds.
         ↓ "那怎么办？"
答案句：The key is to start small — just ten minutes before bed 
        is enough to build the habit.
```

**适用场景**：议论文深化论点、建议信引出对策

---

### 驱动力类型②：观点 → 证据

```
论点句（Claim）给出一个需要被证明的判断
         ↓
信号词：The reason is / Consider the fact that / This is because
         ↓
证据句（Evidence）给出具体、可感知的证据
```

**示例**：

```
论点句：Small daily actions can make a surprisingly big difference 
        to the environment.
         ↓ "你怎么证明？"
信号句：Consider the fact that:
证据句：if every student in Wuhan refused one disposable plastic bag 
        per day, more than two million bags would be kept out of 
        landfills every single day.
```

**适用场景**：议论文支撑论点、书信建议提供依据

---

### 驱动力类型③：现象 → 解释

```
现象句（Phenomenon）描述一个场景或事件
         ↓
推进句：通过感官细节（动作/表情/身体反应）推进画面
         ↓
解释句（Explanation）揭示背后的意义
```

**示例**：

```
现象句：Something unusual happened during our English class last Tuesday.
         ↓ "发生了什么？"
推进句：Just as our teacher was about to begin, she put down her textbook, 
        looked at us, and said: "Today, no grammar."
         ↓ "学生什么反应？为什么这件事值得写？"
解释句：What followed was the most memorable lesson I have ever had — 
        we spent the entire period just talking, in English, 
        and for the first time the language felt genuinely alive.
```

**适用场景**：叙事文推进情节、活动类作文展开过程

---

### 驱动力类型④：抽象 → 具体

```
抽象句（Abstract）给出一个宏观感受/结论/品质
         ↓
驱动词：Specifically / In practice / What this meant was / To be precise
         ↓
具体句（Concrete）将其转化为可被"看见"的行为、画面或数字
```

**示例**：

```
抽象句：That summer, I finally learned what it means to take 
        responsibility for my own learning.
         ↓ "具体做了什么才算'负责'？"
驱动词：Specifically,
具体句：I stopped waiting for my teacher to point out my mistakes 
        and started keeping an error notebook. I also began setting 
        my own weekly goals instead of simply following the 
        assigned schedule.
```

**适用场景**：成长类作文证明"改变真的发生了"、结尾升华前的落地描写

## 衔接词精准选择

不能所有地方都用First/Second/Third。衔接词的选择必须反映句子之间**真正的逻辑关系**。

### 衔接词功能分类表

| 逻辑关系 | 信号 | 精准衔接词 | 不用 |
|---------|------|----------|------|
| **递进**（后者比前者更深/更重要） | 前一论点→更重要的论点 | Furthermore / More importantly / What's more / Above all | Second |
| **转折**（后者与前者方向相反） | 正面→负面 / 预期→意外 | However / Yet / Nevertheless / That said | But |
| **因果**（后者是前者的结果） | 原因→结果 | Therefore / As a result / This is why / Consequently | So |
| **举例**（将前面的抽象观点具象化） | 观点→具体体现 | For instance / Take...as an example / To illustrate | For example（不扣分，但基础） |
| **让步**（承认对方，但不改变立场） | 对方有道理→但 | Although / Even though / While it is true that... | But（不够精准） |
| **总结**（收束段落或全文） | 展开完→收束 | In short / To sum up / Clearly / This demonstrates that | Finally（太弱） |

### 最高频易混淆三组

**① However vs Therefore**

```
However = 方向相反："前面说A，但实际上是B"
Therefore = 方向相同且是结果："前面说A，所以结果是B"

练习：选择正确的一个
"I practiced every day. ______, I passed the exam."
→ 前面练了 → 结果过了（因果）= Therefore ✓

"I practiced every day. ______, I still felt nervous on the exam day."
→ 前面练了 → 但还是紧张（转折）= However ✓
```

**② Although vs Despite**

```
Although + 完整句（有主语+谓语）
Despite + 名词/doing

✓ Although I was tired, I kept going.
✓ Despite being tired, I kept going.
✓ Despite my tiredness, I kept going.
✗ Despite I was tired, I kept going. ← 高频错误！
```

**③ Such as vs For example**

```
Such as + 名词/名词短语（不放句首）
For example, + 完整句（可以放句首）

✓ I enjoy outdoor sports, such as running and swimming.
✓ I enjoy outdoor sports. For example, I run every morning.
✗ I enjoy outdoor sports, for example, running. ← 句式混用
```

## 段落自查清单（精简5条）

写完每个段落后，快速过这五条：

```
□ 1. 有没有哪句话删掉后，前后文逻辑毫无损失？
     → 有 = 孤岛句，需要改造成有机句。

□ 2. 相邻两句之间，我能说出它们是什么逻辑关系吗？
     → 不能 = 缺衔接词或逻辑跳跃。

□ 3. 有没有连续三句用同一个开头？
     → I... I... I... / We... We... We... = 句式单调，至少换一个。

□ 4. 这段的最后一句，是否为下一段的存在提供了理由？
     → 没有 = 段落之间是"并列"不是"递进"。

□ 5. 有没有"举完例子就结束"的句子？
     → 例子后面必须有一句分析/解释（这个例子说明了什么）。
```

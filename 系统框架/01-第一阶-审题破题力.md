# 第一阶：审题破题力

## ——3分钟拿到骨架，不跑题、不空白

---

## 核心问题

学生拿到题目后最常见的三种失败：

```
失败模式①："不知道写什么" → 大脑空白，盯着题目发呆3分钟
失败模式②："写偏了" → 洋洋洒洒写完，发现漏了一个要点
失败模式③："只有一个点" → 写了两句就没话说了，开始硬凑字数
```

这三种失败都有一个共同的根因：**没有一套自动化的"拆题程序"**。

本阶训练的目标就是让这个程序变成肌肉记忆——拿到任何题目，大脑自动跑完三步，3分钟内产出可用的内容骨架。

---

## 核心工具：3问破题法

```
拿到任何中考写作题目，立即运行以下三步：

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

问1：题目要我回答哪几个点？
     → 圈出题目中的"要点关键词"
     → 区分"必须答的点"和"可以加的点"
     
问2：每个点我用什么具体内容填？
     → 禁用空话（important / useful / meaningful）
     → 每个点必须有"可感知的内容"（时间/地点/动作/数字/人名/画面）
     
问3：这些内容我的英语能表达吗？
     → 如果某个想法英语说不出来 → 换一个能说的
     → 宁可写简单的真实内容，不写复杂但错误百出的内容

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 问1的详细操作：要点拆解

中考题目通常给出3-4个要点（有时是编号列表，有时是自然段落）。每个要点 = 你的作文里必须出现的内容。

**示例题目**：

> 上周末你们班去东湖公园开展了"清洁公园"志愿活动。请用英语为校刊写一篇短文（80-100词），介绍这次活动。
>
> 要点：
> 1. 时间和地点
> 2. 参加的人和带的工具
> 3. 活动过程（至少两个动作）
> 4. 你的感受和收获

**拆解结果**：

| 要点 | 类型 | 我准备写什么（用中文先想） |
|------|------|--------------------------|
| 时间+地点 | 必须 | Last Saturday, East Lake Park |
| 人+工具 | 必须 | 全班同学, gloves, bags, tools |
| 动作1 | 必须 | 捡垃圾 (pick up rubbish) |
| 动作2 | 必须 | 拔杂草 (pull out weeds) |
| 感受+收获 | 必须 | 看到干净的路面 → 感觉自豪 |

### 问2的详细操作：内容匹配（禁用空话规则）

这是拉开好学生和普通学生差距的关键一步。普通学生停在了"我知道要写什么方向"，好学生继续追问"具体是什么"。

**禁用空话清单**（写完后自查，出现以下词重新想）：

```
❌ important / useful / helpful / meaningful / unforgettable / wonderful
❌ broaden my horizons / learn a lot / grow up a lot / become better
❌ work hard / try my best / never give up / believe in myself
```

**替换规则**：每个空话必须替换为"可感知的内容"

| 空话 | 可感知的替换（示例） |
|------|-------------------|
| "This experience was meaningful." | "When I looked back at the clean path shining under the afternoon sun, I realized our small hands had really changed something." |
| "I learned a lot." | "I learned that protecting nature starts with people like us." |
| "Reading is important." | "Reading thirty minutes before bed made me calmer before my final exams last year." |
| "My teacher helped me." | "Ms. Lin stayed after class and said quietly, 'Failing once doesn't make you a bad speaker.'" |

**操作口诀**：
> 写完一个句子，问自己："读者能'看见'我这句话里说的东西吗？"
> 如果不能 → 加一个具体的时间/地点/动作/人名/画面。

### 问3的详细操作：可控性筛查

很多学生想在作文里表达很深刻的内容，但英语能力跟不上，结果写出来全是语法错误。

**筛查原则**：

```
想法很好 + 英语能表达 → ✅ 写
想法很好 + 英语表达不了 → ❌ 换一个能说的想法
想法一般 + 英语能准确表达 → ✅ 写（这是考场上的正确选择）
```

**实操**：在草稿纸上用中文列出每个要点的关键词后，快速问自己：

- 这些关键词我知道英语怎么说吗？
- 如果要用过去时写这一段，我会变过去式吗？
- 有没有哪个句子需要用到我没把握的句型？

任何一个答案是"不确定"→ 换内容，不要换句型硬上。

## 工具二：六大主题内容维度速查表

当"3问"中的问2卡住（不知道从什么维度展开），用这张表快速激活思路。

| 主题类型 | 必须覆盖的内容维度 | 可选加分维度 |
|---------|------------------|------------|
| **校园活动** | 时间+地点 / 参与人 / 具体动作(≥2个) / 感受 | 困难→解决 / 同学反应 / 前后对比 |
| **观点建议** | 明确立场 / 理由(≥2个，含具体经历) / 建议 | 让步承认对方 / 名人例子佐证 |
| **成长经历** | 事件+时间 / 困难 / 谁帮了你(具体话/动作) / 学到什么 | 身体反应 / 心理转折细节 / 现在回望过去 |
| **未来计划** | 想成为什么(具体场景) / 原因(真实经历) / 准备行动(≥2个) | 未来画面 / 回扣原因段 |
| **人际关系** | 人是谁 / 发生什么事 / 为什么重要 / 感受 | 一句引语 / 具体帮助的瞬间 |
| **社会现象** | 现象是什么 / 为什么重要 / 正反两面 / 我的立场+建议 | 数据/事实支撑 / 武汉本地元素 |

**使用方法**：拿到题目 → 判断属于哪类主题 → 扫一眼"必须覆盖" → 确保问1拆出的要点都覆盖了 → 从"可选加分"里挑一个最有话说的加进去。

### 速填表演练（配套练习01使用）

**示范**：题目"写信邀请外国朋友来参加学校的运动会"

```
题目类型：☑书信 □发言稿 □短文 □通知

P1 开头段 —— 功能：点题 + 亮立场/建立场景
  我要写：问候 + 我写信是想邀请你来我们学校的运动会
  关键词：excited to invite, school sports meeting, next Friday

P2 主体段 —— 功能：展开核心内容
  分点①：运动会有什么活动（跑步/跳高/拔河）
  支撑细节：tug-of-war is the most exciting, whole class joins
  分点②：为什么你来会很开心（你上次说想体验中国文化）
  支撑细节：opening ceremony, students wear traditional clothes
  分点③（可选）：我们可以一起做什么（带你逛校园/吃热干面）

P3 结尾段 —— 功能：总结+升华+收束
  我要写：希望你能来 + 期待回信
  回扣的关键词（来自P1）：sports meeting → cheer together
```

## 边界意识：识别"绝对不能偏的方向"

中考作文最大的失分不是"写得不好"，而是"写偏了"。以下每类主题的常见偏题方向：

| 主题类型 | 题目让你写 | ❌ 偏题方向 |
|---------|----------|-----------|
| 校园活动 | 介绍一次活动 | 花大量篇幅写"活动前的准备和心情"，活动本身一笔带过 |
| 观点建议 | 表达观点+给建议 | 只写观点没有建议，或建议空洞（"你应该努力"） |
| 成长经历 | 一次具体的经历+收获 | 收获写成口号（"我成长了很多"），没有具体改变 |
| 未来计划 | 我想成为什么+为什么+怎么准备 | 只写"梦想多伟大"，没有准备行动 |
| 人际关系 | 与某个人的具体事件 | 写成"这个人很好"的简介，没有事件 |
| 书信邀请 | 邀请对方来 + 说明为什么 | 只写了活动介绍，忘了"邀请"这个功能 |

**防偏自检**：写完骨架速填表后，对照题目要点逐条打钩。少一个钩 → 回去补。

## 第一阶训练目标

```
通过标准（全部达到才能进入第二阶）：

□ 拿到任何题目，30秒内圈出所有要点
□ 每个要点能在30秒内想出一个"可感知的具体内容"
□ 3分钟内完成骨架速填表
□ 速填表内容不跑题（对照题目要点逐条打钩全部覆盖）

未通过 → 练习01多做一轮，速度提上来再往下走。
```

# 中考英语写作核心能力训练系统

## ——2026武汉中考·三阶递进方案

---

## 系统设计哲学

```
传统路径：拿到题目 → 直接写全文 → 老师批改 → 看分数 → 下一篇
           ❌ 学生始终不知道"我到底哪里不行"

本系统路径：
审题破题 → 意义表达 → 逻辑衔接 → 全文输出
  ✅ 每个阶段独立训练，问题精准定位
  ✅ 练完立刻知道"这个能力我能用了"
```

### 核心信念

> **写作不是"会英语就会写"，而是"会想、会组、会连"。**
>
> 这三件事——想清楚写什么、知道每句话承担什么功能、让句与句产生逻辑驱动力——每一项都可以单独训练。训练到位了，面对任何题目，学生脑子里会自动跑完整个流程。

## 每张微型训练卡的使用方法

每张卡片遵循统一结构，学生自学或教师指导均可：

```
Step 1: 读"功能定位"（30秒）
        → 知道这个表达在文章里"放在哪、干什么用"

Step 2: 背"基础句型库"（2分钟）
        → 3个模板，遮住英文看中文能说出来

Step 3: 学"升级句型库"（2分钟）
        → 2个提分版本，理解"比基础版好在哪"

Step 4: 看"常见错误警示"（1分钟）
        → 提前避开最常犯的坑

Step 5: 做"分级练习"（10-15分钟）
        → Lv1 单句翻译（确保"能用"）
        → Lv2 情境回应（确保"用对地方"）
        → Lv3 段落嵌入（确保"能融入全文"）

Step 6: 看"中考真题链接"（2分钟）
        → 验证"这个功能真的会考"
```

## 80%稳句 + 20%亮句：考场黄金配比

```
考场作文不是炫技场。

一篇中考作文（约80-100词）大约10-12句话：

┌──────────────────────────────────────┐
│ 稳句（8-10句）：                       │
│ 你100%确定正确的主谓宾结构             │
│ 用你最熟悉的时态、最确定的名词单复数    │
│                                        │
│ 亮句（1-2句）：                        │
│ 1个升级句型（如Not only...but also...）│
│ 1个精彩收尾（如Only when...can we...） │
└──────────────────────────────────────┘

⚠️ 一篇文章用太多复杂句 = 风险指数上升
⚠️ 每个亮句写完后问自己："我95%确定这句没错吗？"
   如果不能 → 立刻降级到稳句版本
```

## 自学 vs 教师指导

本系统支持两种使用模式：

**自学模式**：
- 按照21天计划逐日推进
- 每完成一份练习，对照参考答案自评
- 用★评分（见每份练习末尾的评分标准）
- 低于★★★的练习隔天重做

**教师指导模式**：
- 每堂课聚焦1张微型训练卡（约25分钟）
- 剩余20分钟做该卡的Lv3练习 + 学生互评
- 教师重点检查"常见错误警示"里的错误是否再犯
- 每周末做1次综合演练

> **最后提醒：**
>
> 这个系统的目标是"能力内化"，不是"知识浏览"。
>
> 每张卡练完的标准不是"我看懂了"，
> 而是"遮住答案，我能独立写出来"。
